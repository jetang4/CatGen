About Us

This project is a part of the class CSC698. We are all members of the PINC program at SFSU and we all have different levels of coding experience. This project started out as an idea of recreating a current program known as the Genetics Fly Lab. We wanted to create a student learning tool to teach students about genetics in a visually appealing and easily accessible webpage with cats!
